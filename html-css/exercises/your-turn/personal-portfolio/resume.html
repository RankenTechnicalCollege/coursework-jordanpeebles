React CRUD Application Rubric

  • Pick an entity (person, place, or thing) with 4+ properties (e.g., Student: ID, First Name, Last Name, Grad Year)
  • Display a list of entities on page load (5pts)
  • Create a new entity with form controls (10pts)
  • Read individual entities by searching for a property (10pts)
  • Update individual entities (10pts)
  • Delete individual entities (10pts)
  • Save and read entities from local storage (10pts)
  • Utilize Bootstrap for mobile responsive site (10pts)
  • Implement Font Awesome icon(s) (5pts)
  • Apply custom CSS (5pts)

Deployment: App must be deployed to GitHub Pages using gh-pages and source code pushed to the main branch. See instructions below.

GitHub Pages Deployment Instructions

  1. Install gh-pages: npm install gh-pages
  2. Update vite.config.js: set base: '/YOUR REPOSITORY NAME/'
  3. Update package.json:
    • In scripts add: "deploy": "gh-pages -d dist"
    • Outside scripts add: "homepage": "https://github.com/YOUR_USER_NAME.github.io/YOUR_REPOSITORY_NAME"
  4. Build the app: npm run build
  5. Deploy to GitHub Pages: npm run deploy
  6. Commit and push all code to the main branch:
    • git add .
    • git commit -m "React CRUD app initial commit"
    • git push
